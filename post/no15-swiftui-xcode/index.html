不安装macOS10.15的情况下,使用xcode预览SwiftUI

July 30, 2019
SwiftUI
Page content

由于不想测试新系统的诸多Bug,就没有更新到最新的系统,但是又想玩SwiftUI,所以找到了这么一个办法.

首先下载xcode11 Bete版

打开xcode新建一个playground文件,按照SwiftUI的写法写个Text,然后写入最关键代码:

import SwiftUI
import PlaygroundSupport

struct TextP: View {
    var body: some View {
        Text("SwiftUI")
    }
}
// 就是这行代码
PlaygroundPage.current.liveView = UIHostingController(rootView: TextP())
